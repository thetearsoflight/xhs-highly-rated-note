# 批量生成认知爆破文章 Skill

## 执行步骤

### 步骤 1：读取JSON数据

读取 `xhs_career_results.json` 中的所有标题

### 步骤 2：针对每个标题生成深挖指令

对每个标题（主题），生成以下3个深挖指令：

**指令 1**："[主题] 行业痛点 真实案例"
**指令 2**："[主题] 心理学效应 / 哲学名言"
**指令 3**："[主题] 为什么大多数人失败"

### 步骤 3：在knowledge目录深挖

在 `./knowledge` 目录中搜索：
- **找伤疤**：陈思炜与该主题相关的失败经历
- **找SOP**：陈思炜如何解决该痛点
- **找共识谬误**：陈思炜观点与主流观点的不同

### 步骤 4：生成认知爆破文章

使用结构：**大众共识 A -> 现实打脸 B -> 陈思炜结论 C**
- 字数：1100-1200字
- 标题使用原始小红书标题

### 步骤 5：保存文章

- 创建 `articles` 目录（如果不存在）
- 保存到：`articles/关键词_序号_标题_文章.md`

### 步骤 6：完成汇报

告知用户：
- 成功生成的文章数量
- 列出所有保存的文章文件路径

## 输出格式

```
✅ 文章批量生成完成！

📊 统计信息：
- 读取标题数：X 个
- 成功生成文章：X 篇
- 保存位置：articles/

📝 生成的文章列表：
1. xxx.md
2. xxx.md
...
```
