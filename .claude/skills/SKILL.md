# 小红书关键词笔记爬虫 Skill

## 首次使用准备（仅第一次需要）

首次使用前，需要安装依赖：

```bash
# 安装 uv（如果未安装）
pip install uv

# 使用 uv 安装项目依赖
uv pip install --system -r requirements.txt
```

---

## 执行步骤

### 步骤 1：检查登录状态（⚠️ 必须首先执行！）

**重要：在执行任何爬虫操作之前，必须先运行登录检查！**

1. 运行命令：`python check_login.py`
2. 检查输出结果：
   - ✅ 如果显示"已登录" → 继续执行步骤 2
   - ❌ 如果显示"未登录" → **立即停止整个流程**，告诉用户需要先手动登录小红书
3. 只有登录检查通过后才能进行步骤 2

---

### 步骤 2：抓取数据（登录检查通过后才能执行）

- 从用户输入中提取关键词（如"出海"、"职业规划"）
- 提取数量限制（如果没有提供，默认 30 条）
- 运行命令：`python xhs_keyword_spider.py -k "关键词" -n 数量`

---

### 步骤 3：使用 LLM 筛选数据

- 读取生成的 `data/关键词_notes.json` 文件
- **使用 LLM 分析每条笔记的标题**：
  1. 去掉无标题的笔记（标题为空的）
  2. 筛选出与"求职"或"职业规划"相关的笔记
  3. 判断标准：标题中包含求职、职业规划、就业、面试、找工作、工作方向等关键词，或者内容与求职/职业规划高度相关
  4. 保留所有符合条件的笔记

---

### 步骤 4：导出到 Excel

- 使用 openpyxl 创建 Excel 文件
- 保存到：`xhs_career_results.xlsx`
- 包含字段：标题、链接、点赞数

---

### 步骤 5：结果汇报

- 告诉用户原始抓取数据数量
- 告诉用户筛选后保存到 Excel 的数量
- 确认 Excel 文件已生成
